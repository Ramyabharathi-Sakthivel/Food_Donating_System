PROJECT OVERVIEW :

The Food Donating System is a Java-based application developed to reduce food waste and support community welfare by connecting food donors with receivers.
The system allows individuals or organizations to donate surplus food and enables needy users or NGOs to request and receive available food items.
It focuses on simple data management, user interaction, and efficient tracking of donation activities using core Java concepts.

FEATURES :

-->  Donor and Receiver registration
-->  Add and manage food donation details
-->  View available food items
-->  Request or accept food donations
-->  Track donation history and records
-->  Basic validation and error handling
-->  Simple console-based user interaction

TECHNOLOGIES USED :

Java – Core programming language
Object-Oriented Programming (OOP) – Classes, Objects, Inheritance, Encapsulation
Java Collections Framework – ArrayList, HashMap for storing user and donation data
File Handling / Console I/O – For storing and retrieving records (if implemented)
IDE Tools – Eclipse
Database : Oracle
